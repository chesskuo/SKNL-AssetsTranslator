# Manual

1. Find `/data/data/<apk_name>/cache/` on your Android Device.
2. Dump all filename which called `assets_CardXXXXXXXX.zip` to your computer.
3. Move `SKNL-PNGgen` directory to `cache/assets/UI/HDX`.
4. `start.bat` Enjoy~
